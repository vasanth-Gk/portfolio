# 🌐 Vasanth Portfolio Website

## 📌 Project Overview

This is my personal portfolio website built using HTML and CSS.
It showcases my skills, projects, and interests in Artificial Intelligence, Data Analysis, and Engineering.

The website includes:

* About Me section
* Skills (Technical + Soft Skills)
* Project showcase with real project links
* Contact section
* Smooth UI with simple animations

This portfolio represents my journey as an AI student focused on building real-world solutions.

## 🚀 Features

* ✨ Clean and modern UI
* ⌨️ Typing animation for name and roles

* 📁 Project showcase with GitHub and demo links
